# React Learning 

Этот репозиторий содержит мои учебные материалы по изучению React.

## Что я изучаю

- **React** - библиотека для создания пользовательских интерфейсов
- **JavaScript (ES6+)** - современный синтаксис
- **JSX** - комбинация JavaScript и HTML
- **Компонентный подход** - создание переиспользуемых компонентов
- **Props** - передача данных между компонентами

## Технологии в проекте

- React 18+
- JavaScript ES6+
